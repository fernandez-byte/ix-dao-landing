# IX DAO × INFAXIA — Strategic Growth Plan

## North Star Goal

> **Grow INFAXIA users through IX DAO.**
> IX DAO is the growth engine of INFAXIA — not a separate organization.
> Everything IX DAO does exists to bring more users into INFAXIA.

**Decision filter for every action:**
> "Will this bring more people into INFAXIA?" → Yes = Do it. No = Deprioritize.

---

## The Core Loop

```
IX DAO attracts people (Jobs = USDT hook)
        ↓
To get the reward, you must become an INFAXIA user
        ↓
INFAXIA user base grows
        ↓
More INFAXIA users = more IX FP earned = more Jobs completed
        ↓
More Jobs = more people staying in the ecosystem
        ↓
Ecosystem grows → IX DAO attracts more people → cycle repeats
```

---

## Two Entry Points — Bidirectional Funnel

**Entry Point 1 — IX DAO First:**
```
Discovers IX DAO (ads/social)
        ↓
Joins IX DAO Discord Server
        ↓
Gets directed to INFAXIA
        ↓
Registers + participates in events → earns IX FP
        ↓
Uses IX FP to unlock Jobs → earns USDT
```

**Entry Point 2 — INFAXIA First:**
```
Discovers INFAXIA (ads/social/events)
        ↓
Registers + participates in events → earns IX FP
        ↓
Gets directed to IX DAO Discord
        ↓
Joins IX DAO → uses IX FP for Jobs → earns USDT
```

**Connector:** IX FP + Jobs is the bridge between both ecosystems.

## 4 Phases

### Phase 1 — Build
- Discord bot setup (welcome, !mypoints, !jobs, !checkpoints)
- Connect Google Sheet → Discord for IX FP balance queries
- Setup @ix_dao profile (bio, picture, first tweet) + load X credits
- Build IX DAO Jobs page

### Phase 2 — Launch
- "Get Paid to Explore" UX Job campaign — Facebook, Telegram, X
- INFAXIA → IX DAO bridge CTA for existing players inside INFAXIA
- IX DAO Discord welcome message includes steps to earn IX FP via events

### Phase 3 — Sustain
- Weekly Jobs posts, IX FP leaderboard, USDT proof posts
- Auto community activity report → Paul via Telegram
- @ix_dao content: 3x/week (Jobs, highlights, ecosystem updates)

### Phase 4 — Scale
- Announce IXT token plans — Discord only (not public)
- Open LP program to active DAO members first
- Launch referral program (invite friends → both earn)
